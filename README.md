## Description
This project simulates a grocery store system using Java. It demonstrates
object-oriented programming concepts such as inheritance, abstraction,
and interfaces.

Users can view customer information, manage a shopping cart, add or remove
items, and view VIP customer benefits through a menu-driven interface.
